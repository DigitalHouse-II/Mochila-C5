c. Existen muchas formas de usar Git. Por un lado tenemos las herramientas originales de línea de comandos, y por otro lado tenemos una gran variedad de interfaces de usuario con distintas capacidades. La línea de comandos es el único lugar en donde puedes ejecutar todos los comandos de Git. La mayoría de interfaces gráficas de usuario solo implementan una parte de las características de Git por motivos de simplicidad.

a. ¿Quién inventó el sistema de control de versión Git y por qué?

Lo crea Linus Torvalds. La creó porque las distancias opciones que había en su momento(BitKeeper, CVS y Subversion) no cumplian con los requisitos que él necesitaba.


b. ¿A quién pertenece actualmente Github y por qué?

Pertenece a Microsoft. Compran GitHub porque Google adquirió la principal competencia de GitHub: GitLab.

c. ¿Hay otra forma que no sea la terminal para trabajar con Github?

Se pueden subir y descargar archivos pero sin el control de cambios, no se puede aprovechar todo el potencial de los commits.

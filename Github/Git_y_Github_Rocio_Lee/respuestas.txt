A. ¿Quién inventó el sistema de control de versión Git y por qué? Lo crea Linus Torvalds. La creó porque las distancias opciones que había en su momento(BitKeeper, CVS y Subversion) no cumplian con los requisitos que él necesitaba.
B. ¿A quién pertenece actualmente Github y por qué? Pertenece a Microsoft. Compran GitHub porque Google adquirió la principal competencia de GitHub: GitLab.
C. ¿Hay otra forma que no sea la terminal para trabajar con Github?
Solo se puede trabajar desde la terminal con GitHub, ya que se pueden subir y descargar archivos.
